# Email Forge: Email Template Builder

## Overview

Email Forge is a Software as a Service (SaaS) platform providing a comprehensive suite of tools for creating responsive email templates for marketing campaigns and newsletters. It offers a wide range of features, including authentication, email template creation tools, document editing capabilities, file conversion options, and AI-powered image generation.
**Tech Stack:**

- **Framework:** NextJS full stack with App Router and Backend API
- **Styling:** TailwindCSS
- **UI Library:** Shadcn
- **Authentication:** BetterAuth
- **Database:** PostgreSQL with Drizzle ORM

## Features

- **User Authentication:** Secure user registration, login, and profile management.
- **Email Template Creation Tools:**
  - Email template builder with HTML editor.
  - Pre-designed email templates for various industries.
  - Customizable sections, images, text, and buttons.
  - Responsive design for mobile and desktop.
